PROBLEM STATEMENT![image](https://github.com/user-attachments/assets/acf05b6a-2c32-4747-aaba-0cf2a940d9e7)
THE OBJECTIVE OF THIS FINANCIAL PERFORMANCE DASHBOARD PROJECT IS TO ANALYSIS THE FINANCIAL PERFORMANCE OF A COMPANY USING THE DATA.
THIS INTERACTIVE DASHBOARD PROVIDE A COMPRHENSIVE OVERVIEW OF KEY METRIC AND INSIGHTS DERIVED From provided data.
The goal is to create a visually appealing dashboard that provides an overview of the company financial metrics.
![image](https://github.com/user-attachments/assets/1ff5a791-0b07-454b-828d-b4ea7816a55c)
Highlights Of The Dashboard![image](https://github.com/user-attachments/assets/f5462afa-5e3f-46b8-b422-a22bc3651622)
Profit margins by country with detailed breakdowns.
Time trend analysis to track profit growth across months and years.
Top 5 products by profit for improved strategic focus.
Sales by product categories and countries, helping pinpoint key drivers across different periods.

TECHNIQUES USED TO SOLVE THE PROBLEM![image](https://github.com/user-attachments/assets/acf4af48-4276-4ffd-9e32-0e3ccb334a9a)
Dynamic Card Measure With DAX: DAX function are used to create dynamic measure like profit margin and discount in card visuals, providing real time KPIs.
Key Metrics Displayed via Cards: Multiple card visuals show essential KPIs such as Profit Margin and Discount , giving quick insights into business performance.
Interactive Slicers, Filters and Tooltips: Slicers and filters allow users to customize views, while tooltips offer additional context when hovering over visuals.
Dynamic Graphs With Range-Based filtering: Graphs and charts update in real-time based on slicer and filter selections, showing trends and patterns with the chosen data range.
